

##解决滚动元素内有大量dom，造成卡顿问题的方案

[实现的dome效果在线查看][1]



 - 滚动加载，出现无限滚动时，可能给dom过多页面卡死
 - 滚动加载的组件，dom过多，销毁组件的时候也会出现卡死
 - 任何滚动的元素，内部元素过多，都会出现卡顿现象


**针对上面的问题，我们能否有什么优化的方案？**
